## 🛠️ Getting Started

### Prerequisites
- Node.js >= 18.x
- npm >= 9.x

### Installation

```bash
git clone https://github.com/Ph4ntomByte/Summer_Training_Camp.git
cd Summer_Training_Camp
npm install
```

### Development

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production

```bash
npm run build
npm start
```
